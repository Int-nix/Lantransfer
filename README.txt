This is an open source Lan file transfer and chat
run the Lan transfer.py program for a command based approach to file transfer
windows users can use gui.py for a graphical interface

one computer hosts the server and then clients can locate and connect to the server where they can chat with each other and have access to a shared folder
people can 
push filename to the shared folder
pull filename from the shared folder
pushdir to push a folder
pulldir to pull a folder
rmv to remove a file or folder
help only works if it is two command based connections
